`SocialIcons` is used to render one or more social media icons, e.g., in a
footer. You can use [`GitHubIcon`](/components/github-icon) and
[`TwitterIcon`](/components/twitter-icon) or create your own social media icon
with [`Icon`](/components/icon).

## Examples

This example renders social icons with default alignment (start):

```jsx live
<SocialIcons
  values={[
    {
      id: 'github',
      icon: (
        <Link href="https://github.com/454de6e" sx={{ color: 'inherit' }}>
          <GitHubIcon size={48} title="Follow us on GitHub" />
        </Link>
      ),
    },
    {
      id: 'twitter',
      icon: (
        <Link href="https://twitter.com/454de6e" sx={{ color: 'inherit' }}>
          <TwitterIcon size={48} title="Follow us on Twitter" />
        </Link>
      ),
    },
  ]}
  m={2}
/>
```

## System props

`SocialIcons` supports [space props](https://styled-system.com/api#space) and
[color props](https://styled-system.com/api#color) from
[Styled System](https://styled-system.com/).

## Component props

| Name   | Type                       | Default  | Required | Description                             |
| :----- | :------------------------- | :------- | :------- | :-------------------------------------- |
| values | Array of value             |          | ✓        | See below.                              |
| align  | `start`, `center` or `end` | `center` |          | Alignment of social icons (responsive). |

### Value

| Name | Type   | Default | Required | Description                           |
| :--- | :----- | :------ | :------- | ------------------------------------- |
| id   | String |         | ✓        | Key for the underlying list in React. |
| icon | Node   |         | ✓        | Social icon (composition pattern).    |

## Variants

`SocialIcons` does not support variants.
