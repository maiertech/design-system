## Examples

This example shows a header with a few navigation links:

```jsx live
<Header
  title="Header title"
  links={[
    { href: '/blog', text: 'Blog' },
    { href: '/notes', text: 'Notes' },
    { href: '/about', text: 'About' },
  ]}
/>
```

This example shows a header with many navigation links:


## System props

`Header` supports [space props](https://styled-system.com/api#space) and
[color props](https://styled-system.com/api#color) from
[Styled System](https://styled-system.com/).

## Component props

| Name  | Type          | Default | Required | Description                         |
| :---- | :------------ | :------ | :------- | :---------------------------------- |
| title | String        |         | ✓        |                                     |
| links | Array of Link |         | ✓        | Links to be displayed in navigaton. |

### Link

| Name | Type   | Default | Required | Description |
| :--- | :----- | :------ | :------- | ----------- |
| href | String |         | ✓        |             |
| text | String |         | ✓        |             |

## Variants

| Key                | Description                     |
| :----------------- | :------------------------------ |
| `header`           | Set `color` and `bg`.           |
| `header.container` | Set Set `maxWidth` and padding. |

This example shows how to custmize a header with variants:

```jsx live
<ThemeProvider theme={{ header: { color: 'primary', bg: 'background' } }}>
  <Header
    title="Header title"
    links={[
      { href: '/blog', text: 'Blog' },
      { href: '/notes', text: 'Notes' },
      { href: '/about', text: 'About' },
    ]}
  />
</ThemeProvider>
```
