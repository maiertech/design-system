A component to display social media icons and link to the corresponding profile page. Supports [GitHub](https://github.com/) and [Twitter](https://twitter.com/).

## Github

```react
showSource: true;
---
<Provider>
  <Icon type="github" username="mdotasia" />
</Provider>
```

## Twitter

```react
showSource: true;
---
<Provider>
  <Icon type="twitter" username="mdotasia" />
</Provider>
```
